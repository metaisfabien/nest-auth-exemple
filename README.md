## Description

Exemple d'authentification JWT avec [Nest](https://github.com/nestjs/nest) framework.

Congifuration, Mongoose, validation, serialization, Sawagger (en mode dev), Test unitaire, Test e2e...

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Swagger UI

```bash
$ npm run start:dev
```

http://localhost:3000/api
